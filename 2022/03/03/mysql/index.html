命令

命令 作用
use dbname; 选定数据库
show databases; 显示所有数据库
drop database dbname; 直接删除数据库
show tables; 显示默认数据库中的所有表
describe tablename; 表的详细描述
select version(),current_time; 显示MySQL版本信息和当前时间
\c 放弃当前输入的内容
\h 显示命令清单
\q 退出
\s 查看mysql服务器状态信息
